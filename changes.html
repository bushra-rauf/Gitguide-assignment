Explain the process involved in adding code to an existing repository

To add changes to an existing Git repository, follow these steps:

Use the git add command to stage the files or directories you want to include in the next commit. For example:

PS D:\Gitguide-assignment>change-branch

  • Add changes
    • git add .

  • Commit changes:

    • Use the git commit command to create a new commit with a descriptive message. For example:

      git commit -m "Add new feature"
      Use code with caution.
  • Push changes
